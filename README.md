## 주요 기능

**회원가입 없는 간편한 세션 참여**

- 복잡한 가입이나 로그인 절차 없이, 방 번호나 공유 링크만으로 즉시 스터디 룸에 입장할 수 있습니다.

**Supabase 기반 실시간 타이머 동기화**

- `Supabase Realtime`을 활용하여 모든 참여자의 타이머 상태(집중/휴식)가 1초의 오차 없이 동기화됩니다.

**PIP 모드 지원**

- 인터넷 강의나 문서를 볼 때 타이머가 가려지는 불편함을 해소했습니다.
- 브라우저의 PIP(Picture-in-Picture) 모드를 통해 화면 최상단에 미니 타이머를 띄워두고 끊김 없는 집중 상태를 유지할 수 있습니다.

## AI-Driven Development

이 프로젝트는 기획부터 로직 구현, 배포까지의 전 과정에서 AI(gemini-cli)를 적극 활용하여 개발 효율을 극대화했습니다.
